IsolationLevel Type

Namespace: FSharp.Data.Sql.Transactions
Assembly: FSharp.Data.SqlProvider.Common.dll
Base Type: Enum
All Interfaces: IComparable , IConvertible , ISpanFormattable , IFormattable
Kind: Struct
This is an enum type definition

Transaction isolation level for database operations. Corresponds to the System.Transactions.IsolationLevel but provides SQL provider specific functionality.
