ConditionOperator Type

Namespace: FSharp.Data.Sql
Assembly: FSharp.Data.SqlProvider.Common.dll
Base Type: ValueType
All Interfaces: IEquatable<ConditionOperator> , IStructuralEquatable , IComparable<ConditionOperator> , IComparable , IStructuralComparable
Kind: Struct

Represents SQL conditional operators used in WHERE clauses and query expressions. These operators are translated to their SQL equivalents when building database queries.
