AggregateOperation Type

Namespace: FSharp.Data.Sql
Assembly: FSharp.Data.SqlProvider.Common.dll
Base Type: ValueType
All Interfaces: IEquatable<AggregateOperation> , IStructuralEquatable , IComparable<AggregateOperation> , IComparable , IStructuralComparable
Kind: Struct

Represents SQL aggregate operations for GROUP BY clauses and data summarization. Each operation includes the column name to aggregate.
